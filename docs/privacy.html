Privacy Policy

lineTab operates as a browser extension to provide a unique way to begin browsing. This page is used to inform website visitors regarding our policies with the collection, use, and disclosure of Personal Information if anyone decided to use our lineTab. If you choose to use lineTab, then you agree to the collection and use of information in relation with this policy. The Personal Information that we collect are used for providing and improving the Service. We will not use or share your information with anyone.

Storage

lineTab utilizes local storage in order to save basic user preferences including your custom user name, theme options and display options.

Payments

lineTab utilizes the Google Chrome Payments API to process transactions for in-app items, mainly the color theme.

Account Data

lineTab requires access to the user's chrome bookmarks collection in order to view, launch and remove (at the user's choice).
